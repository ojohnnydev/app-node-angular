Para mais informações contate-nos!

Notas:

  • A base de dados utilizada nessa aplicação encontra-se na pasta util/db.sql
  • A ferramenta xampp foi utilizada como servidor para a base de dados
  • Após a configuração da mesma, rode o comando node server.js para se conectar ao mysql
  • Para iniciar a aplicação rode o comando ng serve --open
